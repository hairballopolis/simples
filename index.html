Simples JavaScript Library

What is Simples

Simples is a lightweight javascript library loosely based on jQuery. This library is not intended to replace jQuery, but meant to be a lightweight alternative to the well loved library.

Do you need to do OO JavaScript, make a small website, some basic lightweight library? You are not alone I found I was not using jQuery to its full extent. Enter Simples, taking the compatibility and clean interface of jQuery and turning it into a lightweight just enough library.
